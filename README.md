# SWP391 - Group 1 - Electronics Warehouse Management System (EWMS)

## Project Overview
The Electronics Warehouse Management System (EWMS) is a web-based application
developed using ASP.NET MVC and SQL Server.  
The system aims to support and manage core warehouse operations such as
product management, inventory tracking, stock-in and stock-out processes,
and warehouse reporting.

## Technologies
- ASP.NET MVC
- SQL Server
- Entity Framework
- HTML, CSS, JavaScript
- GitHub
